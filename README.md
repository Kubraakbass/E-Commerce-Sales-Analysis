# 🛒 E-Commerce Sales Analysis

Bu proje, bir e-ticaret satış veri seti üzerinde keşifsel veri analizi (EDA) yaparak satış performansı, kârlılık ve ürün kategorileri arasındaki ilişkileri incelemektedir.

## 🎯 Projenin Amacı

- Satış performansını analiz etmek
- En başarılı ürün kategorilerini belirlemek
- Kârlılık durumunu incelemek
- İndirimlerin kâr üzerindeki etkisini analiz etmek
- Veri görselleştirme ile anlamlı çıkarımlar yapmak

---

## 🛠️ Kullanılan Teknolojiler

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Dataset

Kullanılan veri seti:
**Sample Superstore Dataset**

İçerdiği bilgiler:

- Satış bilgileri
- Ürün kategorileri
- Bölge bilgileri
- Müşteri segmentleri
- İndirim oranları
- Kâr değerleri

---
